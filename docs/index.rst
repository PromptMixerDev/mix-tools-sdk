Welcome to Mix Tools SDK's documentation!
=====================================

Mix Tools SDK is a Python library for interacting with Mix Tools API.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   installation
   usage
   api
   examples

Installation
-----------

.. code-block:: bash

   pip install mix-tools-sdk

Quick Start
----------

.. code-block:: python

   from mix_tools_sdk import Client

   client = Client("your-api-key")
   # Use the client to interact with Mix Tools API
